LAST TRANSACTION WINS

This is a high-stakes Solana transaction game where only the last submitted transaction before the deadline wins the entire prize pool.

HOW IT WORKS:

  • Participants send transactions to the game smart contract
  • Each transaction must include a minimum 0.05 SOL entry
  • The LAST transaction received before the countdown ends wins ALL accumulated SOL
  • Every new transaction extends the countdown by 30 seconds
  • Network conditions may affect transaction timing - no refunds

WARNING: This is a competitive transaction game with real cryptocurrency at stake. Only participate with SOL you can afford to lose. Transactions are irreversible.

For optimal experience, we recommend using a desktop browser with Phantom or Solflare wallet extension.
Mobile devices not supported.
